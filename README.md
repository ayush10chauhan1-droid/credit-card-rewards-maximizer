# Credit Card Rewards Maximizer 💳

A Python project that helps users choose the best credit card for purchases in order to maximize rewards such as cashback or points.

## Features

* Multiple credit card support
* Category-based rewards
* Automatic reward calculation
* Best card recommendation

## Categories Supported

* Dining
* Grocery
* Travel
* Other

## Project Structure

credit-card-rewards-maximizer
│
├── main.py
├── cards.json
├── src/
├── tests/

## Installation

Clone the repository

git clone https://github.com/yourusername/credit-card-rewards-maximizer.git

Move into the directory

cd credit-card-rewards-maximizer

Run the program

python main.py

## Example

Input
Category: dining
Amount: 2000

Output
Card A → ₹100
Card B → ₹40
Card C → ₹60

Best Card → Card A
